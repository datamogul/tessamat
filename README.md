# Postbuilder Assistent für Tessa

Ein RAG chatbot der für SEO und Onlinemarketing genutzt werden kann

## Datenbasis

Nutz eigenes Dataset und sollte gute Antworten geben :)

## Techstack

- **Haystack**: Klassiches RAG pipeline framework
- **Streamlit**: fürs Deployment
- **Sentence Transformers**: Deutsch trainiertes Modell mixedbread-ai/mxbai-embed-large-v1
- **Google Gemini**: Text generation modeli 2.5 Flash
- **InMemory Document Store**: Vektör DB
